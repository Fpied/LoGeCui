# 🛒 LoGeCui

Application mobile de gestion de cuisine avec liste de courses synchronisée en temps réel.

Projet personnel utilisé au quotidien pour gérer mes repas et mes courses.

## ✨ Fonctionnalités

- Gestion de la liste de courses avec quantités et unités
- Synchronisation en temps réel via Supabase
- Interface mobile .NET MAUI (Android)
- Cocher les articles au fur et à mesure des achats

## 🛠️ Technologies

- C# / .NET MAUI (application mobile)
- Supabase (base de données et synchronisation)
- TypeScript
- Inno Setup (installateur Windows)

## 🚀 Installation

1. Cloner le repo
2. Ouvrir `LoGeCui.slnx` dans Visual Studio
3. Configurer les clés Supabase dans le projet
4. Lancer sur un émulateur Android ou un appareil physique
